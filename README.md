# Alzheimer-s-Disease-Project
This project has three clear objectives.
First, to develop a machine learning model that can detect Alzheimer's Disease using EEG signals — making screening early, affordable, and non-invasive.
Second, to design a complete data processing and classification pipeline — covering preprocessing, feature engineering, and model training — to ensure robust and reliable predictions.
And third, to build a Streamlit-based web interface where users can upload EEG feature data and receive real-time Alzheimer's prediction results along with probability scores.
